# کنترلر JetFan

[→ English](README.md)

یک کنترلر هوشمند فن JetFan (تا ۱۳۵۰۰ دور) بر اساس دمای سیستم. آردوینو خروجی PWM رو اعمال می‌کنه و یک برنامه پایتون روی PC دماهای CPU/GPU رو می‌خونه و سرعت فن رو بلادرنگ تنظیم می‌کنه.

## تغییرات نسخه v1 → v2

| ویژگی | v1 (دستی) | v2 (خودکار) |
|-------|-----------|-------------|
| کنترل | دستی via Serial دستورات ۰-۴ + ولوم | خودکار کامل بر اساس دمای سیستم |
| دما | ندارد | می‌خونه CPU (`sysfs`) + GPU انویدیا (`nvidia-smi`) |
| الگوریتم | ۵ حالت PWM گسسته | منحنی دمای پیوسته + PID در تمام بازه‌ها |
| کاهش هوشمند سرعت | ❌ | زیر ۵۰°C هر ۱۰ ثانیه ~۵۰۰ دور کم می‌کنه |
| واکنش نامتقارن | ❌ | افزایش دما → سریع، کاهش دما → آروم (کنترل با ولوم) |
| ولوم حساسیت | کنترل مستقیم PWM | تنظیم قدرت PID (۰=کند، ۲۵۵=لحظه‌ای) |
| ارتباط با PC | ندارد | پروتکل سریال دوطرفه (`PWM`, `SENS`, `RPM`) |
| حالت دستی پشتیبان | ندارد | ۳ ثانیه بی‌فرمانی → ولوم مستقیم PWM رو می‌گیره |
| دوباره وصل شدن خودکار | ندارد | آردوینو رو از `/dev/serial/by-id/` پیدا می‌کنه |
| ذخیره لاگ | Serial monitor | فایل چرخشی + journald + وضعیت یک خطی |

## طرز کار

```
┌─────────────┐  PWM:xxx  ┌──────────────┐
│   آردوینو    │◄──────────│  برنامه PC   │
│  (خروجی PWM) │──────────►│  (پایتون)    │
└─────────────┘ SENS/RPM  └──────┬───────┘
                                 │
                          ┌──────┴───────┐
                          │  دماهای سیستمی│
                          │  CPU + GPU    │
                          └──────────────┘
```

- **آردوینو** ولوم حساسیت (A0) رو می‌خونه و به PC می‌فرسته. PWM که از PC میاد رو اعمال می‌کنه.
- **برنامه PC** دمای CPU (از `sysfs`) و GPU انویدیا (از `nvidia-smi`) رو می‌خونه، با منحنی دما + PID بهترین PWM رو حساب می‌کنه و via سریال می‌فرسته.
- **حالت دستی:** اگه آردوینو ۳ ثانیه فرمان نگیره، ولوم مستقیماً PWM رو کنترل می‌کنه.
- **عدم تقارن:** افزایش دما → فن فوراً زیاد می‌شه. کاهش دما → فن آروم‌تر کم می‌شه.
- **ولوم حساسیت:** ۰ = واکنش کند و میانگین‌گیر، ۲۵۵ = واکنش لحظه‌ای به هر تغییر دما.
- **کاهش هوشمند:** زیر ۵۰ درجه و وقتی دما پایدار بود، هر ۱۰ ثانیه ۵۰۰ دور سرعت فن کم می‌شه.
- **اضطراری:** اگه دمایی از ۸۵ درجه رد شد، فن فوراً ۱۰۰٪ می‌ره.

## نصب

### ۱. آپلود کد آردوینو

فایل `main.ino` رو با Arduino IDE باز کن، بورد و پورت رو انتخاب کن و آپلود کن.

### ۲. نصب برنامه PC

برنامه آردوینو رو از طریق `/dev/serial/by-id/usb-Arduino*` پیدا می‌کنه (فرقی نمی‌کنه به کدوم پورت وصل باشه یا هاب باشه) و خودکار وصل می‌شه.

```bash
chmod +x enable.sh
./enable.sh
```

این کار dependencies رو نصب می‌کنه و برنامه رو به عنوان **سرویس یوزری systemd** ثبت می‌کنه که موقع بوت (حتی قبل از لاگین) اجرا بشه.

### ۳. حذف

```bash
./disable.sh
```

## نیازمندی‌ها

- **سخت‌افزار:** آردوینو (هر مدل با Serial)، JetFan با PWM + تاکومتر، پتانسیومتر ۱۰kΩ
- **نرم‌افزار:** Python 3، `pyserial`، `nvidia-smi` (اختیاری، برای دمای GPU)
- **سیستم‌عامل:** لینوکس (thermal zoneهای sysfs برای دمای CPU)

## پایه‌ها

| قطعه | پایه آردوینو |
|------|-------------|
| PWM فن | D9 |
| تاکومتر فن (ورودی) | D2 (وقفه) |
| تاکومتر (خروجی به لپ‌تاپ) | D7 |
| پتانسیومتر | A0 |
| LED (روی برد) | D13 |

## پروتکل سریال

| جهت | فرمت | توضیح |
|-----|------|-------|
| PC → آردوینو | `PWM:xxx\n` | تنظیم PWM (۰–۲۵۵) |
| آردوینو → PC | `SENS:xxx\n` | مقدار ولوم حساسیت |
| آردوینو → PC | `RPM:xxx\n` | دور فن اندازه‌گیری‌شده |
| آردوینو → PC | `MODE:...\n` | تغییر حالت (AUTO/MANUAL) |
